# 🖼️ CodeAlpha Image Gallery

A modern and responsive Image Gallery web application built using HTML, CSS, and JavaScript.

## 🚀 Features

* Responsive Image Gallery Layout
* Grid-Based Design
* Image Hover Effects
* Smooth Animations
* Image Preview Functionality
* Modern User Interface
* Mobile Friendly Design
* Fast Loading Experience

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

The objective of this project is to create an attractive and responsive image gallery that allows users to browse and view images efficiently while providing a smooth and engaging user experience.
